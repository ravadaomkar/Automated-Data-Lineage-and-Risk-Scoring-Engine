# Assets

Static, imported-by-source assets (as opposed to `public/`, which is served
as-is). Empty for now — add files as the app needs them:

- `images/` — photos, illustrations, screenshots
- `icons/` — custom SVG icons not covered by lucide-react
- `logos/` — brand marks, partner logos
- `animations/` — Lottie/JSON animation files
- `fonts/` — local font files (if not using @fontsource packages)
